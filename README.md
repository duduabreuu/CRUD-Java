Sistema de Cadastro de Pessoas Sistema CRUD desenvolvido em Java para cadastro e gerenciamento de pessoas, criado como trabalho da disciplina de Lógica de Programação do curso de Análise e Desenvolvimento de Sistemas. Funcionalidades

Cadastrar pessoa com nome, ano de nascimento e hobby Listar todos os cadastros com idade calculada automaticamente Buscar pessoa por ID Editar campo específico (nome, ano ou hobby) sem solicitar preencher tudo novamente Remover um registro pelo ID Apagar todos os registros com confirmação Persistência automática em arquivo pessoas.txt

Órgãos transversais

Java Manipulação de arquivos com BufferedReader e BufferedWriter

Como executar

Compile os arquivos:

bashjavac *.java

Execute o programa:

bashjava Principal
